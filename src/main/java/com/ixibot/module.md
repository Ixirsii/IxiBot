# Package com.ixibot.listener

Listeners for message streams which can trigger bot events.

<p>
    Listeners listen to external sources such as Twitch, Discord, or the console then emit
    internal events. This is opposed to Subscribers which listen (or subscribe) to internal
    PubSub events.
</p>
